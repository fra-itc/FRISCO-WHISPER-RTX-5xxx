CLASS HIERARCHY
───────────────

TranscriptionEngine
├── __init__(model_size, device, compute_type, auto_detect_gpu)
├── load_model() → bool
├── transcribe(audio_path, ...) → TranscriptionResult
├── transcribe_with_retry(audio_path, max_retries) → TranscriptionResult
├── get_gpu_info() → GPUInfo
├── cleanup() → None
├── _detect_gpu() → GPUInfo
├── _get_audio_duration(audio_path) → float
├── _format_timestamp(seconds) → str
├── __enter__() → self
├── __exit__() → None
└── __repr__() → str

AudioConverter
├── convert_to_wav(input_file, output_dir, progress_callback) → Path
└── is_wav_compatible(file_path) → bool

TranscriptionResult (dataclass)
├── success: bool
├── output_path: Optional[Path]
├── segments_count: int
├── language: Optional[str]
├── language_probability: float
├── duration: float
└── error: Optional[str]

GPUInfo (dataclass)
├── available: bool
├── device_name: Optional[str]
├── vram_gb: float
├── cuda_version: Optional[str]
├── supported_compute_types: List[str]
└── recommended_compute_type: Optional[str]


USAGE PATTERNS
──────────────

Pattern 1: Simple
    from src.core import transcribe_file
    result = transcribe_file('audio.wav')

Pattern 2: Context Manager (Recommended)
    with TranscriptionEngine() as engine:
        result = engine.transcribe('audio.wav')

Pattern 3: Explicit Control
    engine = TranscriptionEngine()
    result = engine.transcribe('audio.wav')
    engine.cleanup()

Pattern 4: Progress Tracking
    def callback(data):
        print(f"Progress: {data['progress']}%")

    engine.transcribe('audio.wav', progress_callback=callback)
